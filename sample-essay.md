This is a sample essay so you can see how the layout renders, with tags, a thumbnail, and a description.

To add a new essay:
1. Create a new `.md` file in `_essays/`
2. Copy the front matter block above (between the `---` lines) and fill in your own title, date, description, tags, and thumbnail
3. Set `featured: true` only if you want it in the homepage 3x3 grid — otherwise omit that line or set it to `false`
4. Write your essay in markdown below the front matter

Delete this file once you've added your own essays.
